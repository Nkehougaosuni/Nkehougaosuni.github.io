可变和不可变数据类型

不可变数据类型

  • 数字
  • 字符串
  • 元组

    不可变指：地址不变的情况下，其值不可变。

可变数据类型

  • 列表
  • 集合
  • 字典

    可变指：地址不变的情况下，可以改变其内部元素。

赋值

形式

  • 正常的赋值操作：b=a
  • 对象作为参数：f(x)
  • 返回值：return x

解析

以b=a为例，赋值不会开辟新的内存空间，它只是复制了对象的引用。也就是说除了b这个名字之外，没有其他的内存开销。修改了a，也就影响了b，同理，修改了b，也就影响了a。

浅拷贝

形式

  • 可变对象的切片操作：b=a[:]或b=[x for x in a]
  • 工厂函数(数据结构本身的构造器)：b=list(a)
  • copy模块中的copy()函数：b=copy.copy(a)

    不可变对象的切片操作获得的仅仅是对象的引用

解析

浅拷贝，拷贝的程度浅，会重新分配一块内存，创建一个新的对象，但里面的元素是对原对象中各个子对象的引用。

深拷贝

形式

  • copy模块中的deepcopy()函数

解析

深拷贝和浅拷贝对应，深拷贝拷贝了对象的所有元素，包括多层嵌套的元素。因此，它的时间和空间开销要高。
